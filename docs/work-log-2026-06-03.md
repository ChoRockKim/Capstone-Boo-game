## 검증 기록

오늘 작업 중 아래 검증을 사용했습니다.

- `npm run lint`
- `npx tsc --noEmit --pretty false`
- `npx expo export --platform web --output-dir /private/tmp/boo-web-export-audit`

주의:

- lint는 기존 React hooks 계열 오류가 남아 있을 수 있으므로 최신 상태를 다시 실행해 확인해야 합니다.
- 자유투 화면과 마이룸 드래그는 실제 기기에서 체감 QA가 필요합니다.
- 서버 API는 Swagger와 실제 동작이 다를 수 있으므로 배포 전 작은 호출로 다시 확인해야 합니다.

## 배포 전 체크리스트

- 로그인 유저와 게스트 유저 모두 앱 진입이 가능한지 확인
- 로그인 직후 코인, XP, 하트가 서버 값과 맞는지 확인
- 미니게임 시작 시 하트가 1 감소하고, 화면 재진입/서버 동기화 후 튀어 오르지 않는지 확인
- 자유투 5골 성공 시 3코인 지급 알림이 정확한 타이밍에 뜨는지 확인
- 자유투 실패 시 기존 미니게임 스타일의 게임 종료 UI가 뜨는지 확인
- 학식 먹이기 서버 실패 시 로컬 상태가 롤백되고 오류가 보이는지 확인
- 퀴즈 쿨타임이 끝났을 때 버튼이 다시 활성화되고 `00:00:00`이 남지 않는지 확인
- 업적 달성 시 알림이 순차적으로 뜨고 코인/XP 보상이 유지되는지 확인
- 마이룸 부 롱프레스 드래그 시 잡는 순간과 이동 중 위치가 튀지 않는지 확인
- 새 캐릭터 에셋이 registry에 연결되어 실제 화면에서 보이는지 확인
- 웹 export route에 중복/잘못된 route가 생기지 않는지 확인
